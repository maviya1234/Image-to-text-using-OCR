# PaddleOCR Project

A Python-based OCR (Optical Character Recognition) project using PaddleOCR for extracting text from images and documents.

## 📌 Features
- Extract text from images
- Supports multiple languages
- Fast and accurate OCR detection
- Easy to use Python implementation
- Can process JPG, PNG, and PDF files

---

## 🛠️ Technologies Used
- Python
- PaddleOCR
- OpenCV
- NumPy

---

## 📂 Project Structure

├── images/ # Input images  
├── output/ # Extracted text results  
├── main.py # Main OCR script  
├── requirements.txt # Required dependencies  
└── README.md # Project documentation
